## Run Locally

Clone the project

```bash
  git clone https://github.com/HMSOne/HMSOne-Frontend.git
```

Go to the project directory

```bash
  cd HMSOne-Frontend
```

Install dependencies

```bash
  npm install -D pnpm
  pnpm install
```

Start the server

```bash
  pnpm run dev
```
